# pxt-biped

4DOF 2족보행 로봇 확장 블록 (micro:bit + Robotbit)

## 개요

Robotbit 확장보드의 PCA9685 서보 드라이버를 활용하여 4개 서보모터(2힙 + 2발목) 기반 2족보행 로봇을 제어하는 MakeCode 확장 블록입니다.

## 관절 매핑

| 서보 채널 | 관절 | 역할 |
|-----------|------|------|
| S1 | 왼 발목 | 좌우 무게중심 이동 |
| S2 | 왼 힙 | 다리 전후 스윙 |
| S3 | 오른 힙 | 다리 전후 스윙 |
| S4 | 오른 발목 | 좌우 무게중심 이동 |

## 블록 구조

### 초기화
- **로봇 초기화** — PCA9685 초기화 + 홈 포지션
- **보정** — 관절별 각도 보정 (-30° ~ +30°)

### 1층: 관절 제어 (저수준)
- **관절 각도 설정** — 개별 관절 제어
- **포즈 설정** — 4개 관절 한 번에 지정
- **홈으로** — 전체 90° 복귀
- **대기** — ms 단위 대기

### 2층: 동작 프리미티브 (중간)
- **속도 설정** — 느리게/보통/빠르게
- **기울이기** — 발목으로 무게중심 이동
- **스윙** — 힙으로 다리 전후 이동

### 3층: 보행 패턴 (고수준)
- **전진** / **후진** — n걸음 보행
- **회전** — 왼쪽/오른쪽 제자리 회전
- **킥** — 발차기 동작
- **인사** — 숙이기 동작
